# AI-Claims-Copilot-RAG-Powered-Insurance-Policy-Assistant
Built an end-to-end Retrieval-Augmented Generation (RAG) solution that enables users to query insurance and healthcare policy documents using natural language.

Technologies: Python, Streamlit, ChromaDB, SentenceTransformers, Ollama, PDF Processing

Key features include:

PDF document ingestion
Vector embeddings using SentenceTransformers
ChromaDB vector database
Semantic search and retrieval
Ollama-powered local LLM inference
Source attribution and page citations
Audit logging and traceability
Evaluation framework for AI quality monitoring

The system retrieves relevant policy sections and generates grounded answers with supporting citations, making it suitable for insurance, healthcare, compliance, and regulatory use cases.
